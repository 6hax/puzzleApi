Documentação da API Puzzle

Bem-vindo à documentação completa da API Puzzle. Esta API fornece acesso a diversos quebra-cabeças de diferentes categorias para integração em seus aplicativos e jogos.

Introdução

A API Puzzle é uma solução simples para desenvolvedores que precisam de conteúdo de quebra-cabeças para seus projetos. Com endpoints fáceis de usar, você pode obter quebra-cabeças aleatórios ou por categoria específica.

Todas as respostas são retornadas no formato JSON e os endpoints suportam requisições CORS para uso em aplicações web.

Endpoints

GET /randomPuzzle

Retorna um quebra-cabeça aleatório de qualquer categoria disponível no sistema.

Resposta (JSON):
{
  "category": "mathematics",
  "questionObj": {
    "id": 11,
    "question": "Qual é o dobro de 23?",
    "answer": [
      "46",
      "46"
    ]
  }
}
GET /randomPuzzleCategory/:category

Retorna um quebra-cabeça aleatório de uma categoria específica. Substitua :category pelo nome da categoria desejada.

:category string O nome da categoria desejada (ex: "technology", "music")
Resposta (JSON):
{
  "category": "technology",
  "questionObj": {
    "id": 15,
    "question": "Qual foi o primeiro videogame da Nintendo?",
    "answer": [
      "NES",
      "Nintendo Entertainment System",
      "nes"
    ]
  }
}
GET /categories

Retorna uma lista com todas as categorias de quebra-cabeças disponíveis na API.

Resposta (JSON):
{"general, mathematics, technology, literature, geography, history, science, music, ..."}
